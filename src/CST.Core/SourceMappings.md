# PDF to XML Source Mappings

This document records the PDF source mappings implemented in `Sources.cs`. The `startPage` value is the PDF page number where Myanmar page 1 begins (found by locating the large decorative book title heading with "နမော တဿ..." invocation).

**Formula**: `pdfPage = startPage + (myanmarPage - 1)`
